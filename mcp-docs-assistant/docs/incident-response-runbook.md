# Incident Response Runbook

For Sev1 incidents, the on-call engineer must acknowledge the alert within five minutes and create an incident channel immediately.

The incident commander is responsible for stakeholder updates every fifteen minutes until mitigation.

A postmortem is mandatory within three business days after resolution.
