# Remote Work Policy

Employees may work remotely up to three days per week, depending on manager approval and team rituals.

Hybrid teams are expected to be on-site on Tuesdays and Thursdays unless an exception is approved.

All remote employees must keep working hours visible on the calendar and remain reachable through corporate chat during core hours.
